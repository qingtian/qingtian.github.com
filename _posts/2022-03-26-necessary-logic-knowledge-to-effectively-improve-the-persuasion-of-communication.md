# 前言

![1](/img/20220326/1.jpg)

这是我从身边的同事处搜刮而来的一本书，书名和短小是我看中它的关键。

逻辑在我们的日常生活与工作中时常会被应用到，我认为储备一些这类通识知识有百利而无一害。如此有学习价值，又如此短小精悍，明显属于小投入大收益的事，我想你也会忍不住想去这样做吧。

周六，我便躲进蜗牛图书馆里，花费1个多小时，连读带抄地快速阅读完了它。

以下便是我的手抄笔记总结，下面就请你同我一起来看看书中的逻辑学知识吧。

# 我的阅读感受

本书第一章“学习逻辑学的思维准备”，简要介绍了有关事实的基本形式，观念及分类，人类认知以及事物-观念-语言的关系。同时，书中还介绍了有效沟通的基本原则以及方法，解释了真相的两种形态等内容。

第二章“逻辑学的基本原理”，是全书最具价值的内容部分。本章中具体介绍了逻辑学的4个基本原理，灰色地带，逻辑论述的有效方法，以及命题的两种形式。

第三章“逻辑学的语言”，提到最重要的部分即是：论证的两个基本要素（前提+结论）。

全书比较小巧，很快便可读完。通过阅读再去反思自己过往的沟通过程，可以发现很多逻辑不自洽的部分。而通过阅读本书将会成为我未来，提升自身沟通说服力的必经之路。

有关书中第四章“非逻辑思维的根源”及第五章“非逻辑思维的主要形式”的相关内容，在本文中未曾涉及。如果你对这些内容感兴趣的话，建议你自行阅读，细品一翻才好。

# 第一章：学习逻辑学的思维准备

开篇第一节讲的便是：全神贯注。观察是知识的重要来源，注意力是有其价值的。注意力要求我们对所处的环境作出敏捷主动的反应。这就需要我们用心去看，用心去听，学会关注细节。

![2](/img/20220326/2.jpg)

这点就可以联想到《如何成为一个会学习的人》书中所提到的“会学习的秘诀”，同样是把“全身心投入的专注力”作为重要因素。而逻辑思维也正是另一个成就会学习的秘诀。

事实的客观存在有两种基本形式，即：**事物** + **事件**。事物即是存在的实体，如动物，物品等。事件是由事物组成的，或由事物的表现形式组成，如最近火爆的俄乌战事。

一般来说，**要确认事物的存在，我们只需要实地去考察**。比如，要证明中国北京的确实存在，那就去北京旅行，亲眼看看它，这是最直接也最可靠的方法。而有时，我们无法通过亲身体会来确认，比如林肯被刺之类的事件。那么我们就只能**通过一些可作为间接证据的事物来确认此事件的真实性**。

**观念是对客观事物的主观反映**。其主要包括两种类型：

- 正确观念：指反映其对象的客观秩序。
- 错误观念：指反映其对象的歪曲秩序。

![3](/img/20220326/3.jpg)

我们只能通过观念来了解世界，但观念只是人类认知的工具，而非最终目的。最有效的确认观念正确与否的方法，是透过观念本身去观察其所表现的对象。

这里有一个新的概念，叫“人类认知”。将观念联系事实，组成了人类认知的3个部分：

- 客观存在的事物。
- 事物在大脑中的反映。
- 我们为其创造的语言（借之我们才能与他人交流）。

关于“人类认知”，你可以考虑读一读尤瓦尔·赫拉利写的畅销书——《人类简史》。有关认知革命、农业革命、科学革命的故事，可以帮助我们更好的认知人类与世界。

书中提到的观念，其包含两种不同的类型：

- **简单观念：指符合我们大脑中的定义**。
- **复杂观念：指与那些客观事物并非一一对应的观念**。

从以上的一些基本概念，你可以大致感受到事物，观念，语言的关系。**事物是客观的存在，观念是客观事物的主观反映，而语言是我们与他人沟通的工具**。观念与语言之间紧密切合，是顺畅交流的关键。

![4](/img/20220326/4.jpg)

沟通最基本的第一步是：**语言与观念的匹配**。语言要忠实表达出客观事物的本来面貌，从而使我们的沟通有坚实的事实基础。而“有效沟通”的基本原则便是：

```
不要想当然地认为你的听众会领悟你没有直接表达的意思。
```

如果我们希望做好与他人的有效沟通，书中也给出了4种方法。它们分别是：

- 说完整的句子。
- 不要将主观看法当作客观事实。
- 避免使用双重否定。
- 根据对象选择合适的语言。

如果究其根源的话，制约有效沟通的典型因素有2个，即：**模糊** + **多义**。通常，一个词语使用越普遍，它的含义就越模糊。而多义词也就是在逻辑学中所说的“双关语”。因此要避免产生歧义的方法是：

```
让你所运用的词语尽可能有针对性地反映出你的本意，
以便读者或者听众不用费心去猜测你所说的到底是什么。
```

所有的逻辑推理，所有的论证，目的只有一个：找出某个事物的真相。真相是我们所有努力的意义所在。最近发生的东航坠机空难，到底真相是什么？这是我们真正关注的重点内容，而不是那些借此炒作的“投机者们”。

**真相存在两种形态，一种称为本体真相，一种称为逻辑真相。本体真相指的是关乎存在的真相，逻辑真相便是逻辑学家关注的真相形式**。

最后，正如美国联邦最高法院大法官“路易斯·D·布兰代斯”所说：

```
语言逻辑应该以事实逻辑为基础。
```

此时，我不由地想起儿时所看《名侦探柯南》动画片中的那句口头禅：“真相只有一个”。无论如何掩盖也无法以假乱真，事实与真相永远只有一个。


# 第二章：逻辑学的基本原理

逻辑学可以被认为是一门科学、一门艺术，或者一项技能，它也完全可以被认为是这三者的综合体。

![5](/img/20220326/5.jpg)

书中指出逻辑学有4个基本的原理，它们分别是：

- **同一律**：事物只能是其本身。
- **排中律**：对于任何事物在一定条件下的判断都要有明确的“是”或“非”，不存在中间状态。
- **充足理由律**：任何事物都有其存在的充足理由。
- **矛盾律**：在同一时刻，某个事物不可能在同一个方面既是这样，又不是这样。

其中，矛盾律是逻辑学中最关注的原理。关于基本原理还有其他一些值得一提的东西。**首先，基本原理是不证自明的**。当我们看到这些基本原理时，直觉反应一定是“当然是这样”。**其次，基本原理的另一大特点是，它是不能被证明的**。因为基本原理反映的是绝对基础的事实，它们是人类意识行动的首要基础。

**灰色地带是真相不能被清晰认出来的情况**。工作中也常常存在这类灰色地带，例如：一个项目的工作被拆分出不同的需求任务，而涉及的相关人员人数众多。一个工作项要不未明确一位负责人，要不就没有一位整体的负责人。此时，该项目就出现了所谓的“灰色地带”。

权责的不明确容易导致项目风险出现时，无法找到一位担责人。除了问题的解决会耗费更多的时间外，还会导致团队内出现工作推诿的现象，这是一个管理者需要特别警惕的问题。**我认为处理“灰色地带”有效的解决方法就是：1人主负责制**。由一位能力匹配的人员承担项目负责人，全面领导与管理、跟进与处理过程中遇到的任何问题。对这个项目完全负责，沟通集中化，权责集中化，有问题就找他的一种定位。

**在逻辑论述中，避免语义不清和模棱两可最有效的方法就是：定义术语**。定义术语主要分为两个步骤，即：

- 将要定义的术语放入最相近的类别当中。
- 确定其与同类中其他事物的不同特性。

如果联系实际工作，术语的定义往往会出现在技术方案中。每位技术同学的背景，经验不足，对于特定业务领域的业务术语，那就非常有必要提前定义与在团队中对齐。这是有效避免在沟通过程中出现错频的方法。这也是为什么我特别关注领域知识储备的原因。财务，人力等业务领域，会有非常多的专业术语。如果我们不提前储备，那在“约哈里窗户理论”的作用下，我们是很难做到互相理解的。**做好提前储备业务专业知识+不耻下问是我认为比较有效的应对方法**。

这里有另一个名词，叫做：“直言命题”。那什么是直言命题呢？一般来说，**最有效的论证，其结论都是直言命题，即清楚明确地告诉我们事物的真相是什么**。但有时，一个命题形式上是直言命题，但实际上它表达的内容仍然可能不对，比如告诉我们的只是讲话者坚定的信念，而不反映任何客观情况。

除了直言命题，还有一种命题叫做：“普遍命题”。它涵盖的对象非常广。使一个普遍命题成立要满足两种条件：

1. 它所陈述的事物是真实的。
2. 适用于整个类别。

明确的语言表达在普遍命题中很重要，因为它能避免听众产生困惑的可能。普遍命题有两种形式：**全称命题** + **特称命题**。全称肯定命题意味着“所有的”，“每个”，它肯定了某个类别的所有事物的某种共性。特称命题则不对其类别的所有个体发生作用。通常它会被定语“一些”所限定，但无论部分是大是小，部分都只能是部分。

# 第三章：逻辑学的语言

逻辑推理的基本步骤，即推理的过程，就是根据已知正确的第一个观点，推断出第二个观点，而第二个观点之所以正确，是由于第一个观点的正确。推理的过程构成了论证的核心。

论证是由命题组成，推理所关注的观点是由命题来表达的。从本质上来讲，每个论证都有两个基本要素，也就是两个不同类型的命题：**前提**  + **结论**。

**前提是一个支持性命题，它是一个论证的起点，包含推理的出发点所依靠的基础事实。结论是被证明的命题，它在前提的基础上得出，并为大家所接受**。

实际上，我们应该尽量避免从一个论证出发得出多个结论的情况。单一确定的结论总是最好的，换言之，**最有效的论证总是试着得出最简单明了的结论**。

对于这个论证/逻辑推理的应用，我觉得日常中最经常用到的地方，便是工作汇报。

例如，领导给你安排了一个调研的活。对于领导来说他只关注，用哪个更好。而想要得出这个更好的结果，本质上便是基于哪些事实来进行逻辑推理。基于不同解决方案的机遇与挑战来进行分析，基于大环境的实际情况进行分析，最终得出调研结论。当领导与你的意见相左时，想要提升你所提供方案的说服力的话，往往就需要借助于底层逻辑了。这是一种逻辑思维的能力，是以客观事实为依据的话术技巧。

![6](/img/20220326/6.jpg)

**所谓向上管理的重点在于工作汇报，而工作汇报的重点在于思维逻辑**。这只是我看到的一个示例，逻辑学的应用远不止这些，学以致用还需要我们善于分析问题 + 勤于思考。正如那句俗话所说：办法总比困难多。

在整个逻辑学的语言中，还包括很多内容，例如：

- 从全称到特称
- 从特称到全称
- 断言
- 否定命题
- 比较
- 正确论证
- 条件论证
- 三段论
- 归纳论证
- 评定论证

这里我就不一一举例说明了，本文主要是给你一个直观感受，如果对这些感兴趣，可以好好地读一读《简单的逻辑学》这本书。其实以上的内容会和“离散数学”中“命题逻辑”的相关知识有交叉，这些数学层面的知识就得你自己发挥知识搜索能力去挖掘了，我在此就不多加赘述了。

# 后记

随着本书阅读的不断深入，我就越发感觉自己过去，与他人工作沟通中的逻辑问题。也因此更加感受到了学习逻辑学知识的必要性。通过富有逻辑性的论述，可以更加有效地表达个人观点。通过术语的定义，解释与双向理解，可以有效地与他人沟通。通过符合MECE原则的前提命题陈述，可以更加有利地说服对方接受己方的想法或意见。

个人沟通表达上的逻辑自洽，重要性不言而喻。对于最近正在进行的CPP答辩而言，内容，逻辑与表达是答辩的三个重要方面。“逻辑”就是其中一个非常重要的方面，由此也可见“逻辑”的地位之高，我们去学习与掌握它的意义也就更大了。

最后，我想说的是：

```
要想沟通更有说服力，肚子里没点逻辑学知识还真不行。
```

从现在开始，让我们一起且学且珍惜，且学且应用吧~

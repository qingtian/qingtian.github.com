# 前言

眼看暑假已过半，而我还并没有带贝贝去外面逛过。这样下去怎么行，当然不行，怎么办，当然是去外面逛逛。去哪里呢?最近我在迷博物馆————中国的11大博物馆。而在杭州周边比较近的就要属上海和南京了。

南京博物院是我向往的地方，无奈它要提前1周预约，只得作罢。既然目的选定了南京，那么我就开始通过马蜂窝里寻觅值得一去的地方。一一记录在本子上作为待去目的地，而最后我就从中规划了本次2日游的主要目的地了。

它们分别是：明孝陵、中国科举博物馆、南京城墙博物馆。至于为什么没有选定总统府，中山陵，原因只有一个——因为太热门，我根本没约到。

第一次带着家人去南京，还得先出发再说吧，毕竟真正做到比什么都重要。

# 说走就走的行程

从周五晚上开始谋划，到最后成行，我花了大概3个小时抢票，定酒店，看攻略并制定初步行程。虽然从最后的结果看，本次行程并不美好，甚至有些痛苦。

但是，人生就是这样，如果不去做，我们永远都无法真正了解想做的与真正做之间的差别。尽管经历了痛苦，但对于我们真正能走出来，来到南京就已是一个巨大的飞跃。

南京挺好，以后可以常来来。

使用手机备忘录制定的初步行程：

![20230805_1](http://img.qingtian16265.com/20230805_1.jpeg)

从复盘角度来看，针对带有两个小孩的家庭来说，1天游2个景点足矣，再多就会显得疲惫。从夏天的气候来看，适合出行的时间点是早晨+晚上太阳下山后，清晨凉快人少，晚上有夜景显得与众不同。

要说我们为什么没从早上就出发，主要是在于早上6：00 - 8：00 有地龙战队晨练训练营的活动。每个组织都有纪律，纪律性，时间性是需要家长以身作则用行动告诉小朋友的道理。这样比起用言语表达会显得更有效果和说服力。

晨练后，贝贝在一鸣吃了面包，而我买了拌面吃。运动过后，这种身体与心灵的双重满足，真是难得的享受。和暑假其他几次训练一样，我抽空也剪成了视频，可以看我B站的短片：[<地龙战队晨练训练营06>](https://www.bilibili.com/video/BV19h4y1r75E/?spm_id_from=333.999.0.0)。我可不吝啬拍照与视频，我最渴望与锻炼的就是自己的摄影与剪辑能力，实践是检验真理的唯一标准。

# 12306买票的故事

买火车票当然是12306，但因为正值暑假，南京又是热门城市，买票的活就变成了抢票的事。去程的票还算好，为了保证训练不耽误，又留下足够时间让贝贝完成必要的作业，我选择在12：03的高铁，基本1个半小时便可以到达南京南站。

而回程的票就比较难搞了。要么早上太早，要么就下午和晚上都没票，综合考虑后，我只得选择了11：35的高铁。这么来看，我所谓南京2日游，实际真正在南京游玩的时间连24小时都不到，说本次旅行是一次特种兵之旅，我是一点也没夸张。甚至还带了才1.5岁的小儿子，还得时刻用小推车代步。期间的艰辛只有真正经历的人才能明白，不说也罢。

这里我要说的一个故事是：原以为这样的行程已经很奇葩了，可是当我周日清晨醒来查看短信时，发现12306给我发了6条短信，内容分别是：山东地震了，原回程高铁班次取消了。

当我看到这条消息时，我的内心是崩溃的。要知道之前抢票已经不易，这么一出之后，我们能不能周日顺利坐火车回家都是一个未知数。幸运的是，我抢到了12：10的动车票，这才免去了徒增费用与时间的尴尬。不过代价是，目的地是杭州西站（仓前），而不杭州东站，坐地铁的时长多了差不多半小时，火车上的时间也多了1个小时。时间变长了，路途变远了，但回家的目的达到了。

# 定酒店

说走就走最大的两项内容便是上面的抢票，以及现在说到的定酒店。以前的酒店预定都是内人来操作，今天这次由于小儿子内人在哄睡，只能由我来处理了。

酒店千差万别，考虑到带着小儿子一起出行，我就打算选个相对省事一点的。价位稍高的，距离景点稍近的，说人话就是：少折腾些的。本来打算直接定到秦淮河景区里的民宿，但看到最后的房间无窗，塌塌米，小又临街吵闹，遂作罢。最后，选在了南京大学附近的鼓楼晶丽酒店，2大2小定了一个1.8米的大床房。

![20230805_2](http://img.qingtian16265.com/20230805_2.png)

![20230805_4](http://img.qingtian16265.com/20230805_4.png)

酒店附送了水果，夜宵券和代金券，而且在我们周六下午入住后，免费帮我们升级成为了套房。整个体验下来还是非常不错的，套房外是一个客厅有独立的电视，四人座椅和桌子。

![20230805_5](http://img.qingtian16265.com/20230805_5.png)

里面是张大床，电视，书桌，贵妃椅，还附带两个卫生间，一个大浴缸可以用来泡澡。这个项目对于贝贝真是美极了，在他看来，游玩后能够泡个澡真的美极了。虽然周六逛的很晚，但贝贝依然和小睿两个泡了会热水澡，看他们玩水开心的样子，想必是高兴坏了。

选定晶丽的几个原因是：

- 靠近地铁站，南京打车不太方便，还是地铁靠谱时间可控
- 距离钟山景区与秦淮河景区属于之间，可以两个点走两条线，不重复

![20230805_3](http://img.qingtian16265.com/20230805_3.png)

# 出发

今天正好爸爸要回宁波治疗，所以我们一行4人和爸爸一起坐地铁去火车东站。10点40左右出发的，大概也就花了40分钟左右，我们便到了火车站。因为检票口相距很远，所以在地铁出站后，我们便和爸爸分开了。他去往宁波，我们去往南京。

![20230805_6](http://img.qingtian16265.com/20230805_6.png)

出发早，中饭也没来得及吃上一口。进站后，内人和小朋友就在检票口处等待，而我被派去做一项任务——搞定午饭食物。时间已经临近发车时间，什么KFC，麦当劳都被我直接PASS，屏幕里显示的等待叫号量就让我明白此处没戏，等不起。最后，我在德克士选了一个鸡腿汉堡的三人餐打包搞定。

到检票口时正好开始排队检票，我们在暑假大军顺利进站。

![20230805_9](http://img.qingtian16265.com/20230805_9.png)

虽然高铁晚了几分钟，但无伤大雅，在座位上吃着午饭，看着车窗外的沿途风景，这才是暑假正确的打开方式。

# 路途中的事

我和内人的座位并未并排，我在啃枪腿的同时，一边的美女车友在用手机定餐。

![20230805_7](http://img.qingtian16265.com/20230805_7.png)

![20230805_8](http://img.qingtian16265.com/20230805_8.png)

一份牛肉面，一份米饭。多年以来并没坐过高铁，车上也已经有了一些变化。我观察来看，主要是这么几点：

- 点餐更方便，点完会送餐到座位，便民。
- 车上有了更多的商业销售活动：乳酪，糖，饮料等会不断地来来回回，当然还有盒饭。
- 列车组乘务员走动的也更加频繁，在检查行李架的同时，也会关注每个人所需，及时提供帮助。

![20230805_17](http://img.qingtian16265.com/20230805_17.png)

本次路途中，最大的发现便是：江苏境内，竟然在农田处有很多大风车。

![20230805_10](http://img.qingtian16265.com/20230805_10.png)

我猜应该是风力发电设备吧，心里不禁嘀咕：难道是江苏盛产台风，风力很强的缘故？不然怎么会用上风力发电呢（一片迷惑）。

# 第一站：明孝陵

大概经过1.5小时后，我们顺利到达南京南。从高铁出发，就可以看到地铁引导标识，我查看了一下地图后发现打车耗时比地铁还多，因此我英明地选择了更好的地铁方式。我们第一站的目的地是：明孝陵。

![20230805_13](http://img.qingtian16265.com/20230805_13.png)

![20230805_12](http://img.qingtian16265.com/20230805_12.png)

在地铁站里，给本次配戴墨镜的贝贝先来上一张，再交代一下所处的地铁站台环境，这种摄影思维与短片剪辑有异曲同工之妙。特定，中景，远景，分别交代人物情绪，从近及远地详细描述周边环境，这个手法很好用。

从南京南到明孝陵需要先上3号线，再经过8站到达大行宫。

![20230805_18](http://img.qingtian16265.com/20230805_18.png)

![20230805_11](http://img.qingtian16265.com/20230805_11.png)

再转2号线，经过3站到达苜蓿园站，再步行7分钟即可到达明孝陵3号门售票处。

![20230805_19](http://img.qingtian16265.com/20230805_19.png)

下了地铁后才切实感受到南京的热情，这种热情温度接近40度。要不是得益于南京街头高耸的梧桐树的话，我想气温可以直接让我们焉了。户外活动想都不敢想，可出生牛犊不怕虎，我们的第一站却偏偏选择了广阔的陵园。这只是本次旅途的第一个败笔，教训深刻。

![20230805_14](http://img.qingtian16265.com/20230805_14.png)

![20230805_15](http://img.qingtian16265.com/20230805_15.png)

明孝陵，中山陵都位于钟山风景区。门口是一个游玩区域，有儿时熟悉的玩具车，还有近年来火热的一个凹凸墙，贝贝在这还特意向我申请耗时5分钟顶着大太阳玩了一小会儿。小朋友就是这样，天热不可怕，可怕的是不好玩。

![20230805_16](http://img.qingtian16265.com/20230805_16.png)

随着人游我们一路推着小睿向景区门口走去，蜿蜒的小径，高大遮蔽的树木，柏油路面，外加路边的河流构成了明孝陵给我的第一印象。当然，还有这些景观造型的小人正招手欢迎远方的来客，不知我们是否在列，我想应该在的。

![20230805_20](http://img.qingtian16265.com/20230805_20.png)

![20230805_21](http://img.qingtian16265.com/20230805_21.png)

明孝陵的门票是成人70元，1.4米以下免票。

![20230805_22](http://img.qingtian16265.com/20230805_22.png)

3号门口有景区的导览图，从这里检票进去的人也算不多，不知是否是因为天气太热。不一小会，我们便检票入陵了，开始真正感受这陵园的风光与魅力。

![20230805_23](http://img.qingtian16265.com/20230805_23.png)

当然，首当其冲的是不吝啬拍照，尽管用的是手机，但也要留下尽可能多，尽可能美的影像素材。作为文章配图，短片片断，也是非常美好的回忆。真实的存在，真实的发生，真实的记忆，不加修饰，就是这么的纯净。

![20230805_24](http://img.qingtian16265.com/20230805_24.png)

而贝贝永远是我成为摄影师之路上的头号模特，哈哈哈。
